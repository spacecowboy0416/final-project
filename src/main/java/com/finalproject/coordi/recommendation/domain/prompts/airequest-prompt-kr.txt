첨부된 이미지 아이템을 메인 아이템으로 사용하고 전신 코디를 구성하세요.

- fit: slim|regular|relaxed|oversized|wide
- item category: top|pants|outerwear|shoes|accessory|dress|jacket
- coordination slot keys: tops|bottoms|outerwear|shoes|accessories
- priority: essential|optional

작업 지침:
1) 첨부된 이미지를 분석하고 디자인, 색상, 소재, 핏을 포함한 메인 의류 아이템을 식별하세요.
2) 환경 컨텍스트(온도, 날씨, 현지 시간, 요일)를 고려하여 보온 수준과 격식을 결정하세요.
3) 나머지 슬롯에 적합한 아이템을 추천하여 코디를 구성하세요.
4) 현재 날씨에 편안하고 전체 스타일의 일관성을 보장하세요.
5) 코디의 모든 필수 슬롯을 채우세요.

필수 제약사항:
- 출력은 AiBlueprintConstraintDto 구조와 정확히 일치해야 합니다.
- coordination은 모든 슬롯을 포함해야 합니다: tops, bottoms, outerwear, shoes, accessories.
- 각 슬롯은 다음을 포함해야 합니다: item_name, search_query, category, attributes, temp_range, reasoning, priority.
- temp_range는 정확히 두 개의 정수 [min, max]여야 하며 min <= max입니다.
- category는 item category 코드를 사용해야 합니다 (top|pants|outerwear|shoes|accessory|dress|jacket).
- attributes.color/material/fit/style은 허용된 enum 코드를 사용해야 합니다.
